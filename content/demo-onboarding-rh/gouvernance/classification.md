# Classification des données (contenu de démonstration)

Le cadre distingue quatre niveaux de données : **publique**, **interne**,
**personnelle**, **sensible**.

Règle de la V1 : un harnais du cadre ne traite que les niveaux **publique** et
**interne**. La présence de données **personnelles** ou **sensibles** rend le cas
inéligible en l'état et impose un renvoi vers le DPO.

| Source | Niveau |
|---|---|
| Règlement du temps de travail (SRC-001) | interne |
| Mutuelles labellisées (SRC-002) | interne |
| Note télétravail (SRC-003) | interne |
| Fiche marchés publics (SRC-004) | publique |
| Contacts utiles (SRC-005) | interne |
| Premiers jours (SRC-006) | interne |

Aucune source du corpus ne contient de donnée personnelle réelle : les contenus
sont entièrement fictifs. La classification est validée par le DPO avant tout
usage réel.
