# Classification des données (contenu de démonstration)

Le cadre distingue quatre niveaux de données : **publique**, **interne**,
**personnelle**, **sensible**.

Règle de la V1 : un harnais du cadre ne traite que les niveaux **publique** et
**interne**. La présence de données **personnelles** ou **sensibles** rend le cas
inéligible en l'état et impose un renvoi vers le DPO.

|---|---|

| Protection sociale complémentaire (SRC-002) | interne |
| Note de service — télétravail (SRC-003) | interne |

| Contacts utiles par fonction (SRC-005) | interne |

| Plan de formation (SRC-010) | interne |
| Organigramme fonctionnel (SRC-011) | publique |
| Politique de sécurité des SI (SRC-012) | interne |

Aucune source du corpus ne contient de donnée personnelle réelle : les contenus
sont entièrement fictifs et relèvent du **Syndicat mixte du Val de Brenne**
(organisation fictive). La classification est validée par le DPO avant tout usage
réel.
